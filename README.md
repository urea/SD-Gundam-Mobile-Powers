# モビルパワーズ Mobile Powers App

トレーディングカードゲーム「モビルパワーズ」をPC/スマホのブラウザで遊びやすくする、モビルパワーズ用ファンメイドWebアプリです。

- 公開URL: https://sd-gundam-mobile-powers.vercel.app
- 現行版: v1.0.0

## 概要

カード確認、デッキ編成、構築済みデッキ、CPU対戦をまとめて扱えるWebアプリです。スマホでも表示・操作できるように調整しています。

このアプリは非公式のファンメイド作品です。公式作品・権利元とは関係ありません。

## 主な機能

- ルールブック表示
- カードビューワ
- デッキ編成
- デッキコード生成/読込
- 構築済みデッキ
- ルールベースCPU対戦
- PC/スマホ対応

## 使い方

1. 公開URLをブラウザで開きます。
2. `カードビューワ` でカードを確認します。
3. `デッキ編成` でデッキを作成、または構築済みデッキを読み込みます。
4. `CPU対戦` で対戦を開始します。

## バージョン管理

このプロジェクトはSemVer風に管理します。

- `1.0.0`: 第一回公開版。
- `1.0.x`: 誤字修正、表示崩れ修正、軽微なカードデータ修正。

- `2.0.0`: デッキコード互換性やゲーム進行仕様など、利用者に影響する破壊的変更。

変更履歴は [CHANGELOG.md](./CHANGELOG.md) に記録します。

## 開発

前提:

- Node.js

セットアップ:

```powershell
npm install
```

開発サーバー:

```powershell
npm run dev
```

ビルド:

```powershell
npm run build
```
