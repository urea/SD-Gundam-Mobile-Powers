# モビルパワーズ Webアプリ

トレーディングカードゲーム「モビルパワーズ」のカード確認、デッキ編成、CPU対戦ができる非公式ファンメイドWebアプリです。

https://sd-gundam-mobile-powers.vercel.app

## 概要

ブラウザ上でカードを確認し、デッキを編成し、構築済みデッキや自作デッキでCPU対戦できます。PCとスマホのどちらでも利用できます。

## できること

- ルールブックを確認する
- カード一覧を見る
- デッキを編成する
- デッキコードを生成・読み込む
- 構築済みデッキを読み込む
- CPUと対戦する

## 使い方

1. 公開URLをブラウザで開きます。
2. `カードビューワ` でカードを確認します。
3. `デッキ編成` でデッキを作成、または構築済みデッキを読み込みます。
4. `CPU対戦` で対戦を開始します。

## 対応環境

- PCブラウザ
- スマホブラウザ
